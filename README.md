## 「一般」模式注意事項
### (一) 採集
- 採集為複選，每次派出隨機選擇採集種類。
- 如果只想挖農場，只勾選農場。
- 「隊伍數量」表示要派出的隊伍數量/滿隊。
- 例子：小號只夠派1隊，滿隊2隊，則顯示「1/2」；主帳能派5隊，滿隊5隊，則為「5/5」。
